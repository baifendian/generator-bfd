## yeoman平台百分点项目生成器

### 使用

```sh
$ npm install --g yo

$ npm install --g generator-bfd

$ cd workspace

$ yo bfd myapp
```

如果在项目目录已存在，在当前文件下创建

```sh
$ npm install --g yo

$ npm install --g generator-bfd

$ cd myapp

$ yo bfd
```
